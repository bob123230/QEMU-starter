# QEMU-starter
这是一个用来启动QEMU的程序。

**运行说明：**

<p>1.安装PYTHON环境。</p>
<p>2.打开 QEMU启动器.py 。</p>

**！注意**

<p>1.一定要将QEMU-starter的根目录设为程序运行目录。</p>
<p>2.请不要修改项目数据！</p>
